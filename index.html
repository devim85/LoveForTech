Web Protocols Overview

HTTP - Hypertext Transfer Protocol

HTTP is the foundation of data exchange on the Web and a client-server protocol used for fetching resources like HTML documents.

HTTPS - HTTP Secure

HTTPS is the secure version of HTTP, where communications are encrypted for security.

FTP - File Transfer Protocol

FTP is used for the transfer of computer files between a client and server on a computer network.

TCP - Transmission Control Protocol

TCP is known for its reliability in data transmission. It ensures that all data packets are delivered in the correct order and without errors. This is particularly important for video streaming where the quality of the content is a priority.

Pros of TCP for Video Streaming:

  • Reliability: Guarantees that video packets arrive intact and in the correct sequence.

UDP - User Datagram Protocol

UDP is a simpler, connectionless protocol used for situations where speed is more critical than reliability. For live video streaming or gaming, UDP is often preferred.

Pros of UDP for Video Streaming:

  • Speed: Faster data transmission since it does not wait for acknowledgments.

WebSockets

WebSockets provide a full-duplex communication channel over a single, long-lived connection, which is ideal for real-time applications like video conferencing.
